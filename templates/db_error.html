🔧 Quick Fix — 3 Steps

Step 1 — Create your .env file

Copy .env.example to .env in your project folder:

copy .env.example .env
Step 2 — Set your MongoDB Atlas URI

Open .env and add your MongoDB Atlas connection string:

MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/sarathi_db # Get this from: MongoDB Atlas → Connect → Drivers → Python

🆓 MongoDB Atlas Free Tier: mongodb.com/atlas — M0 cluster is completely free.

Step 3 — Install dnspython (for mongodb+srv://)

Run this in your project terminal:

pip install dnspython>=2.6.0 # Or install all dependencies: pip install -r requirements.txt
